# Software Test Engineer - Automation

This project demonstrates a complete test automation workflow using Selenium WebDriver, including:

- Test Strategy & Planning
- Automated Test Script Development
- Modular Test Automation Framework
- CI/CD Integration with GitHub Actions
- Bug Reporting & Process Improvement

## Technologies

- Python
- Selenium WebDriver
- Pytest
- GitHub Actions
- Page Object Model (POM)
